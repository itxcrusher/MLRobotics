Project assignments approved software list

This is the list of approved sources of software for your projects. The list is partly based on my own suggestions, and partly based on software which students have requested and been allowed to use in previous years. If you would like to use something which is not already on this list, please email me to get it approved and added. If you use software which is not on this list without getting approval first, that will negatively affect your marks.

Please remember that there is always a price to be paid when you use other people's software (including mine). You have to demonstrate your understanding of it by describing its underlying models and key algorithms. A failure to pay this price will negatively affect your marks.

  • Any Sandbox code which I have provided, and any packages which we have used with it (e.g. numpy, scipy, matplotlib).
  • Pandas, sklearn, seaborn
  • NEURON (https://neuron.yale.edu/neuron/static/docs/neuronpython/firststeps.html Links to an external site.)
  • Brian (https://briansimulator.org Links to an external site.)
  • NEAT-Python (https://neat-python.readthedocs.io/en/latest/ Links to an external site., https://github.com/CodeReclaimers/neat-python Links to an external site.)
  • DEAP (https://deap.readthedocs.io/en/master/tutorials/basic/part2.html, Links to an external site. https://github.com/deap/deap Links to an external site.)
  • CTRNN (https://github.com/madvn/CTRNN Links to an external site.) and 
  • NetworkX (https://networkx.org Links to an external site., https://pypi.org/project/networkx/ Links to an external site.)
  • PyTorch (https://pytorch.org Links to an external site.). I have yet to hear a very good project proposal involving PyTorch - you can use it in principle, but straightforward machine learning applications are not a good fit for this module.
  • Keras (https://keras.io Links to an external site.). The same applies to Keras as for PyTorch. 
  • TensorFlow (https://www.tensorflow.org Links to an external site.). The same applies to TensorFlow as for PyTorch and Keras.
  • CoppeliaSim (https://coppeliarobotics.com Links to an external site., https://coppeliarobotics.com/downloads Links to an external site.)

  • ARGos (https://www.argos-sim.info Links to an external site.)
  • Mesa (https://mesa.readthedocs.io/en/latest/ Links to an external site.)
  • The Hebbian learning implementation described here:
      • https://medium.com/deepgamingai/hebbian-learning-over-reinforcement-learning-for-game-intelligence-4043792ee34a Links to an external site.
